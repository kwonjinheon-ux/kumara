# Colink.nz Membership Levels

Colink.nz membership levels must improve convenience, speed, and capacity. They
must not block the core community experience.

Core principle:

```text
Level difference = convenience difference, not access denial.
```

All members should keep the same basic access to community features. Higher
levels add better organization, faster workflows, richer alerts, less friction,
and better selling tools.

## Levels

| Level | Emoji | Points | Positioning |
| --- | --- | ---: | --- |
| Iron | ⚙️ | 0 | Basic member experience |
| Silver | 🥈 | 300 | Slightly more capacity and light alerts |
| Gold | 🏅 | 900 | Better organization and seller convenience |
| Platinum | 💠 | 1,800 | Power-user search and bulk management |
| Diamond | 💎 | 3,200 | Faster workflows and stronger visibility |
| Master | 👑 | 5,200 | Premium convenience and recommendations |
| Grandmaster | 🏆 | 8,000 | Advanced convenience and high-capacity alerts |
| Challenger | 🌟 | 12,000 | Highest convenience, strongest UX, largest capacity |

My Page should show:

- Current level with the matching emoji
- Current points
- Progress bar toward the next level
- Remaining points to the next level
- Full level benefits table

## 2-1. Keyword Alerts

키워드 알람 수는 회원등급에 따라 자연스럽게 확장한다. 기능 접근을 막는 방식이
아니라 더 많이, 더 편하게 관리할 수 있게 하는 차이다.

| Level | Keyword alert limit |
| --- | ---: |
| Iron | 5 |
| Silver | 8 |
| Gold | 12 |
| Platinum | 20 |
| Diamond | 30 |
| Master | 50 |
| Grandmaster | 80 |
| Challenger | 100 |

이 표는 `내 페이지 → 회원등급 / 포인트` 탭에서 전체 등급 혜택과 함께 보여준다.
`키워드 알람 설정` 탭에서는 현재 등급, 등록 가능 수, 현재 등록 수만 간단히 보여준다.

## 10. Seller Analytics

Platinum and above:

- View count
- Saved count
- Click-through rate
- Basic selling performance summary

## 11. Soft Community Visibility

Diamond and above:

- Subtle post border emphasis
- Slightly higher top-placement probability

This should stay soft. Colink.nz should avoid aggressive pay-to-win placement.

## Highest Impact Features

1. Saved-post folders
2. Scheduled bumping
3. Saved search filters
4. Price-change alerts
5. Seller analytics

## Development Prompt

```text
Korin의 회원 등급 시스템은 기능 제한이 아니라 사용자 경험 향상을 중심으로 설계한다.

모든 회원은 동일한 기본 기능을 사용할 수 있도록 하며, 등급이 높아질수록 찜 개수 증가, 찜 폴더 기능, 가격 변동 알림, 저장된 검색 필터, 자동 끌어올리기 예약, 다중 글 관리, 빠른 채팅 기능, 광고 제거, 맞춤 추천 등 편의 기능이 점진적으로 추가되도록 한다.

회원 등급에 따른 차이는 사용자가 스트레스를 받지 않도록 제한 중심이 아닌 편의성과 효율성 중심으로 설계한다.

특히 찜 기능은 단순 개수 증가를 넘어서 폴더 관리 기능을 포함하고, 검색 기능은 저장 및 빠른 접근 기능을 제공하여 사용자의 탐색 경험을 개선한다.

판매자에게는 조회수, 관심수, 클릭률 등의 간단한 분석 기능을 제공하여 거래 효율을 높인다.

전체 시스템은 경쟁보다는 자연스러운 참여 유도와 사용자 경험 향상을 목표로 한다.
```
