# Korin 내 페이지 구조

로그인 후 상단 메뉴의 **“내 계정보기”**는 **“내 페이지”**로 변경합니다.

```text
로그인 후 메뉴
- 내 페이지
- 알림
- 로그아웃
```

---

# 1. 내 페이지 전체 메뉴 구조

```text
/my-page

내 페이지
├── 프로필 설정
├── 내가 쓴 게시글 관리
├── 내가 쓴 댓글 관리
├── 내가 찜한 글
├── 1:1 채팅방
├── 비밀번호 및 보안
└── 회원등급 / 포인트
```

# 2. 프로필 설정

회원가입 때 입력하지 않은 정보를 나중에 수정할 수 있게 합니다.

## 수정 가능 항목

```text
프로필 사진
닉네임
자기소개
사는 지역
세부 지역
카카오톡 ID
전화번호
연락처 공개 여부
알림 설정
비밀번호 재설정
```

## 프로필 이미지 기능

```text
- 사용자의 컴퓨터/휴대폰에 저장된 이미지 업로드
- 둥근 프로필 이미지로 표시
- 이미지 확대/축소 가능
- 이미지 위치 조정 가능
- 저장 시 정사각형으로 crop
- 화면에는 원형 프로필로 표시
```

## 프로필 설정 화면 예시

```text
프로필 설정

[둥근 프로필 이미지]
[이미지 변경]
[크기 조절 슬라이더]

닉네임
[입력창]

자기소개
[입력창]

사는 지역
[도시 선택]

세부 지역
[입력창]

카카오톡 ID
[입력창]

전화번호
[입력창]

연락 방식
[✓] Korin 1:1 채팅 허용
[ ] 카카오톡 ID 공개
[ ] 전화번호 공개

[저장하기]
```

# 3. 내가 쓴 게시글 관리

사용자가 작성한 모든 게시글을 관리할 수 있게 합니다.

## 주요 기능

```text
- 카테고리별 보기
- 전체 보기
- 검색
- 선택 삭제
- 전체 선택 삭제
- 게시글 수정
- 판매글 상태 변경
```

## 화면 구조

```text
내가 쓴 게시글

[전체] [중고거래] [구인구직] [자유게시판] [생활정보] [부동산]

[검색창]

[ ] 전체 선택

[ ] 중고거래 | IKEA 책상 판매합니다 | 판매중 | 조회 120 | 관심 8
[ ] 자유게시판 | 해밀턴 생활 질문 | 조회 45 | 댓글 3
[ ] 구인구직 | 파트타임 구합니다 | 조회 80 | 댓글 1

[선택 삭제]
```

중고거래 글은 여기서 바로 상태 변경도 가능하게 합니다.

```text
[판매중] [예약중] [판매완료]
```

# 4. 내가 쓴 댓글 관리

사용자가 작성한 댓글을 카테고리별로 확인하고 삭제할 수 있게 합니다.

## 주요 기능

```text
- 카테고리별 댓글 보기
- 원문 게시글 바로가기
- 댓글 선택 삭제
- 전체 선택 삭제
```

## 화면 구조

```text
내가 쓴 댓글

[전체] [중고거래] [자유게시판] [생활정보] [구인구직]

[ ] 전체 선택

[ ] 자유게시판 | "저도 이 정보 궁금했어요."
원문: 해밀턴 병원 추천 부탁드립니다

[ ] 중고거래 | "아직 판매 중인가요?"
원문: 삼성 모니터 판매합니다

[선택 삭제]
```

# 5. 내가 찜한 글

찜한 글은 카테고리별로 정리해서 관리할 수 있게 합니다.

## 주요 기능

```text
- 전체 찜 목록 보기
- 카테고리별 보기
- 선택 삭제
- 전체 선택 삭제
- 상세 페이지 바로가기
```

## 화면 구조

```text
내가 찜한 글

[전체] [중고거래] [부동산] [구인구직] [업체정보]

[ ] 전체 선택

[ ] 중고거래 | IKEA 책상 | $80 | Hamilton | 판매중
[ ] 부동산 | 플랫메이트 구함 | Auckland | 주 $250
[ ] 구인구직 | 카페 파트타임 | Hamilton

[찜 삭제]
```

# 6. 1:1 채팅방

Korin 내부에서 사용자끼리 직접 연락할 수 있게 합니다.

## 주요 기능

```text
- 채팅방 목록
- 읽지 않은 메시지 표시
- 게시글 기반 채팅
- 사용자 프로필 클릭 시 채팅 시작
- 차단 기능
- 신고 기능
```

## 채팅방 목록 예시

```text
1:1 채팅방

[프로필] KiwiSeller
IKEA 책상 문의
마지막 메시지: 오늘 픽업 가능할까요?
2분 전

[프로필] SeoulNZ
삼성 모니터 문의
마지막 메시지: 아직 판매 중입니다.
1시간 전
```

# 7. 게시글 작성자 프로필 → 채팅 연결

모든 게시글에는 작성자 정보가 표시됩니다.

## 게시글 작성자 카드

```text
[프로필 이미지]
닉네임: KiwiSeller
등급: Gold
작성시간: 2시간 전
조회수: 120
관심수: 8

[1:1 채팅하기]
```

## 작동 방식

```text
사용자가 작성자 프로필 클릭
→ 작성자 프로필 미니 카드 열림
→ [1:1 채팅하기] 버튼 표시
→ 채팅방 생성 또는 기존 채팅방으로 이동
```

단, 본인 글에서는 채팅 버튼을 숨깁니다.

```text
본인 게시글일 경우:
[내 글 관리] [상태 변경] [수정]
```

# 11. 개발 프롬프트 최종본

```text
Korin의 로그인 후 사용자 메뉴 이름을 ‘내 계정보기’가 아니라 ‘내 페이지’로 변경한다.

내 페이지는 프로필 설정, 내가 쓴 게시글 관리, 내가 쓴 댓글 관리, 내가 찜한 글, 1:1 채팅방, 비밀번호 및 보안, 회원등급/포인트 탭으로 구성한다.

프로필 설정에서는 회원가입 시 입력하지 않았던 항목들을 나중에 자유롭게 추가, 수정, 삭제할 수 있게 한다. 사용자는 프로필 사진, 닉네임, 자기소개, 사는 지역, 세부 지역, 카카오톡 ID, 전화번호, 연락처 공개 여부, Korin 1:1 채팅 허용 여부를 수정할 수 있어야 한다.

프로필 사진은 사용자의 기기에 저장된 이미지를 업로드할 수 있게 하며, 둥근 이미지로 표시한다. 사용자는 이미지 크기를 확대 또는 축소하고 위치를 조정할 수 있어야 하며, 저장 시 정사각형으로 crop된 뒤 원형 프로필 이미지로 사용된다.

내가 쓴 게시글 관리 탭에서는 사용자가 작성한 모든 게시글을 카테고리별로 확인할 수 있어야 한다. 전체 선택, 부분 선택, 선택 삭제 기능을 제공한다. 중고거래 글의 경우 수정 페이지에 들어가지 않아도 판매중, 예약중, 판매완료 상태를 바로 변경할 수 있어야 한다.

내가 쓴 댓글 관리 탭에서는 사용자가 작성한 댓글을 카테고리별로 확인할 수 있어야 한다. 각 댓글은 원문 게시글로 이동할 수 있어야 하며, 전체 선택, 부분 선택, 선택 삭제 기능을 제공한다.

내가 찜한 글 탭에서는 사용자가 찜한 게시글을 카테고리별로 확인하고 관리할 수 있어야 한다. 전체 선택, 부분 선택, 찜 삭제 기능을 제공한다.

1:1 채팅방 탭에서는 사용자가 진행 중인 모든 채팅을 확인할 수 있어야 한다. 채팅방은 게시글 기반으로 생성되며, 읽지 않은 메시지 표시, 차단, 신고 기능을 포함한다.

모든 게시글에는 작성자의 프로필 이미지, 닉네임, 회원 등급, 작성시간, 조회수, 관심수가 표시되어야 한다. 사용자가 글쓴이의 프로필을 클릭하면 프로필 미니 카드가 열리고, 1:1 채팅하기 버튼을 통해 채팅을 시작할 수 있어야 한다. 단, 본인이 작성한 글에서는 채팅하기 버튼을 숨기고 내 글 관리 버튼을 표시한다.

비밀번호 및 보안 탭에서는 비밀번호 재설정, 이메일 인증, 2단계 인증, 모든 기기에서 로그아웃 기능을 제공한다.

디자인은 Figma 스타일의 세련된 회사 서비스 화면처럼 구성한다. 연한 회색 배경, 흰색 카드형 레이아웃, 둥근 모서리, 부드러운 그림자, 깔끔한 입력창, 명확한 버튼, 넉넉한 여백을 사용한다. 데스크톱에서는 왼쪽 사이드 메뉴와 오른쪽 콘텐츠 구조를 사용하고, 모바일에서는 상단 탭 또는 드롭다운 메뉴로 변경한다.
```
