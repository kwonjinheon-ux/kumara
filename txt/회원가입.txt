아래처럼 **가입은 최대한 짧게**, 상세 정보는 **가입 후 선택 입력**으로 나누는 것이 좋습니다.

## 1) 빠른 회원가입 폼 — 필수 최소

| 항목            | 필수 여부 | 설명              |
| ------------- | ----: | --------------- |
| 이메일 또는 휴대폰 번호 |    필수 | 로그인 ID          |
| 비밀번호          |    필수 | 최소 8자 권장        |
| 닉네임           |    필수 | 커뮤니티 표시 이름      |
| 약관 동의         |    필수 | 이용약관, 개인정보 처리방침 |
| 만 14세 이상 확인   |    필수 | 커뮤니티 안전 기준      |

**추천 문구**

```text
회원가입

이메일 또는 휴대폰 번호 *
비밀번호 *
닉네임 *

[ ] 이용약관 및 개인정보 처리방침에 동의합니다. *
[ ] 만 14세 이상입니다. *

가입하기
```

## 4) 개발용 프롬프트

```text
Korin 회원가입은 최대한 빠르게 완료되도록 설계한다.

초기 회원가입 필수 항목은 이메일 또는 휴대폰 번호, 비밀번호, 닉네임, 약관 동의, 만 14세 이상 확인으로 제한한다.

이메일 인증, 현재 지역, 세부 지역, 프로필 이미지, 자기소개, 관심 카테고리, 알림 설정, 거래 가능 지역, 사업자 여부, 휴대폰 인증, 거주 상태 등은 가입 후 선택 입력 항목으로 제공한다.

회원정보를 더 많이 입력할수록 추가 기능이 열리도록 설계한다. 예를 들어 이메일 인증을 완료하면 중고거래 글쓰기와 찜 기능을 사용할 수 있고, 휴대폰 인증을 완료하면 신뢰 배지와 거래 기능이 확장된다. 사업자 인증을 완료하면 구인공고와 업체 홍보 기능을 사용할 수 있다.

가입 단계에서는 사용자의 이탈을 줄이는 것을 최우선으로 하며, 상세 정보는 마이페이지에서 단계적으로 입력하도록 한다.
```
